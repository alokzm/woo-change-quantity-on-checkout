Customers can change quantity on checkout page. 

== Description ==

This plugin allow you to change the quantity & delete the product on the checkout page.

== Installation ==

* Login to your WordPress dashboard and navigate to Plugins > Add New
* Click on Upload Plugin button. Then browse the 'woo-change-quantity-on-checkout.zip' file from your system
* Click on Install Now.
